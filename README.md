# Void Rush

© Contributors — Licensed under CC BY 4.0

Void Rush is the reference open-source game world built on the Aetheris engine. It serves as both a high-fidelity demonstration of the engine's capabilities and a starting point for community-built worlds.

## Documentation

Game design documents and world specifications are available at: [docs.aetheris.io/void-rush](https://docs.aetheris.io/void-rush)
